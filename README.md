# RPA & Automation Adoption Survey — Power BI Analysis

## Context

Analysis built for a graduate thesis (Mastère Spécialisé, Grenoble École de Management)
examining how professionals across industries adopt automation tools like Power Automate,
UiPath, and low-code platforms.

Sample: n = 41 respondents across multiple countries, roles, and industries.

## Key Finding

Low-code tools (e.g., Power Automate) show significantly higher adoption
rates than enterprise-grade platforms, driven by accessibility and
lower training barriers.
